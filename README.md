# Java Fundamentals

Early Java exercises written while learning the language in 2023. These cover the core building blocks of object-oriented programming — the concepts that everything else in this portfolio builds on.

Included here for transparency about where I started and because some of these, even as small programs, required working through real logic problems rather than just copying syntax.

---

## What's Here

### Object-Oriented Basics
Small programs exploring class construction and object behavior: `Dog Object`, `TV Object`, `Student Object`, `Midterm Student Object`. Each creates a class with fields, constructors, getters/setters, and methods, then instantiates objects and calls behavior.

### Inheritance
A progression of inheritance exercises building from simple to more complex: `Animal Inheritance`, `2 Variable Inheritance`, `Multi-Level Inheritance`, `4 Level Inheritance`, `Class E Inheritance`. The later ones chain several levels of parent/child classes and explore how fields and methods are inherited or overridden down the chain.

### Constructors & Overloading
`Default Constructor`, `Overloaded Constructor`, and `Method Overloading` demonstrate how Java handles multiple ways to initialize an object and call methods with different parameter signatures.

### Control Flow
`Sum For Loop`, `Multiples of 3`, `Largest Number Identifier`, `Factor30` — basic loop and conditional logic. Factor30 finds all factors of 30 using modulo arithmetic.

### Arrays & Vectors
`Manipulating an Array`, `Array Passing to a Method`, `5 Element Vector`, `Midterm Example Vector`, `Final-1 Vector` — cover array traversal, passing arrays as method parameters, and working with `Vector` as a resizable collection.

### Polymorphism, Abstract Classes & Interfaces
`Polymorphism`, `Abstractism`, `Interface`, `Final-Abstract class` — demonstrate how Java handles runtime behavior through method overriding, abstract class contracts, and interface implementation.

### Exception Handling
`Dividing Zero Error`, `Final Exceptions`, `Final Exception Correction`, `Error Explanation` — cover `try/catch` blocks, common exceptions like `ArithmeticException`, and how to handle and recover from errors gracefully.

### Other
`Pitcher Methods` — a method-focused exercise modeling a baseball pitcher's stats. `Part-Time Method`, `Eat Method`, `Pep Rally Code`, `Non-Access Keyword` — miscellaneous method and keyword exercises. `Electronic Payment System-OOJ` — an early OOP simulation of a payment processing system with inner classes modeling payment methods and transactions.

---

## Context

These were written in IntelliJ IDEA during an introductory Java course in late 2023. The jump from these to the projects in the other repositories happened over roughly two years of continued coursework and independent building.
