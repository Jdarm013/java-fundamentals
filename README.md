# Java Fundamentals

Early Java exercises written while learning the language in 2023. These cover the core building blocks of object-oriented programming — the concepts that everything else in this portfolio builds on.

Included here for transparency about where I started and because some of these, even as small programs, required working through real logic problems rather than just copying syntax.

---

## What's Here

### Object-Oriented Basics
`Dog Object` — a simple class with fields, constructors, getters, and methods. `Electronic Payment System-OOJ` — an early OOP simulation of a payment processing system with inner classes modeling payment methods and transactions.

### Inheritance
A progression of inheritance exercises building from simple to more complex: `Animal Inheritance`, `2 Variable Inheritance`, `Multi-Level Inheritance`, `4 Level Inheritance`, `Class E Inheritance`. The later ones chain several levels of parent/child classes and explore how fields and methods are inherited or overridden down the chain.

### Constructors & Overloading
`Default Constructor` and `Overloaded Constructor` demonstrate how Java handles multiple ways to initialize an object.

### Control Flow
`Sum For Loop`, `Multiples of 3`, `Factor30` — basic loop and conditional logic.

### Arrays & Vectors
`Manipulating an Array`, `Array Passing to a Method`, `5 Element Vector`, `Final-1 Vector1`, `Final-makeVector` — cover array traversal, passing arrays as method parameters, and working with `Vector` as a resizable collection.

### Polymorphism, Abstract Classes & Interfaces
`Abstractism`, `Final-Abstract class` — demonstrate how Java handles runtime behavior through method overriding and abstract class contracts.

### Exception Handling
`Dividing Zero Error`, `Final Exceptions`, `Final Exception Correction` — cover `try/catch` blocks, common exceptions like `ArithmeticException`, and recovering from errors gracefully.

### Other
`Part-Time Method` — models employee types using inheritance across full-time, part-time, and intern classes. `Non-Access Keyword` — explores Java non-access modifiers. `Random Goodwill` — uses `Math.random()` to generate randomized goodwill messages.

---

## Context

These were written in IntelliJ IDEA during an introductory Java course in late 2023. The jump from these to the projects in the other repositories happened over roughly two years of continued coursework and independent building.
